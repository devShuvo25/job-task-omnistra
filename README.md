# 🌌 Chargeflow-Inspired SaaS UI Kit

A premium, developer-centric UI kit built with **React**, **Tailwind CSS**, and **Framer Motion**. This project recreates the sophisticated "space-grade" aesthetic of modern fintech and chargeback automation platforms, featuring radial burst graphics, technical dotted grids, and high-performance layouts.

## ✨ Key Features

### 🛠️ Technical Design System
* **Radial Burst Backgrounds:** Custom SVG-generated starburst effects with 120+ animated lines.
* **Dotted Grid Architecture:** Precise `border-dotted` layouts used for footers and section dividers to mimic technical blueprints.
* **High-Contrast Dark Mode:** Deep `#050505` backgrounds with `#2141F3` (Chargeflow Blue) accents.

### 🧩 Premium Components
* **Interactive 404 "Orbit" Page:** A centered, motion-enhanced error page optimized to avoid React hydration errors (uses stable indexing instead of impure functions).
* **Dotted Footer:** A complex, multi-column footer with platform signup integrations and compliance badges (SOC 2/GDPR).
* **Request Integration CTA:** A high-impact newsletter section with a responsive radial burst graphic and hover-scaling effects.
